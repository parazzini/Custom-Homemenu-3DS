How to install

Installation guide

How it works — this mod is installed with LayeredFS, a Luma3DS feature that displays the files from the SD card instead of your console, you can remove it at any time without issues

Prerequisites

  • Luma3DS custom firmware 10.3 or newer — install or upgrade it here
  • Luma's game patching enabled: boot the console holding SELECT to open the Luma3DS configuration, tick "Enable game patching", then press START to save and exit

Download

  1. Download your preferred archive from below and uncompress it
  2. Choose your console region and copy/paste the luma folder in the root of your SD card

How to remove it

Uninstall

  1. Boot the console holding START to launch GodMode9
  2. Navigate to the pathSDCARD:/luma/titles
  3. Delete the folder based on your region:
